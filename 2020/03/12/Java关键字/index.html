Java 关键字和对象

Author: Yannis Date: March 12, 2020  13:07:52 Category: Java

关键字

this

Java中的this用来指向自己的引用。

super

Java中用super来实现对父类成员的访问，用来引用当前对象的父类

static（存放在data seg）

  • 在类中，用static声明的成员变量为静态成员变量，针对于某个成员变量
  • 用static声明的方法为静态方法，不能访问非static的成员（不能访问非静态成员）
  • 可以通过对象引用或类名访问静态成员

语句

package 和 import 语句

Java引入包（package）机制，提供类的多重类命名空间，可以用来管理大型软件系统中数目较多的类，解决类的命名冲突问题。

命名格式

确保类得class文件位于正确路径下

主要的包介绍
java.lang 包含一些Java的核心类，eg:String、Math、Integer、System、Thread
java.awt 包含构成抽象窗口工具集
java.applet 包含applet运行所需的一些类
java.net 包含执行与网络相关操作的类
java.io 包含能提供多种输入和输出功能的类
java.util 包含一些二实用工具，eg:定义系统特性、使用与日期日历相关的函数。
  • 如果将一个类放入一个包中，在该类得源文件第一句写package
  • 执行一个类需要写全包名

Tips:

类名. 都为静态

字符串常量分配在data seg
