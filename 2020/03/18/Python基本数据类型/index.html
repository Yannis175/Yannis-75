Python 基本数据类型

Author: Yannis Date: March 18, 2020  16:02:08 Category: Python

数字类型

整数类型

$$
pow(x,y)用来计算x^y
$$

可以嵌套使用

浮点数类型

$$
e=a*10^b
$$

复数类型

设z=a+bj

对于复数z,用 z.real 和 z.imag 表示它的 实数 和 虚数 部分

内置操作符

​ + - * /

操作符 意义
x//y x与y的整数商
x%y 取余
-x x的负数
+x x本身
x**y x的y次幂

内置函数

函数 意义
abs(x) 绝对值
divmod(x,y) （x//y,x%y）输出为元组类型
pow(x,y[,z]) （x**y）%z
pow(x,y) x**y
round(x[,ndigits]) 对x四舍五入，保留ndigits位小数
round(x) 返回四舍五入的整数值
max(x1,x2,…,xn) 最大值
min(x1,x2,…,xn) 最小值

内置转换函数

函数 意义
int(x) 将x转换为整数
float(x) 返回浮点数x或者字符串x所对应的整数类型
complex(re[,im]) 生成复数，re为实部，im为虚部

字符串操作符

操作符 意义
x+y 连接字符串
x * n 或者 n * x 复制n次字符串x
x in s 如果x是s的子串，返回True,否则返回False
str[i] 索引，返回第i个字符
str[N:M] 切片，返回索引第N到第M的子串，其中不包含M
